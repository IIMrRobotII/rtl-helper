Privacy policy

RTL Helper · Last updated 2026-05-31

RTL Helper runs locally in Chrome. It fixes right-to-left page text on sites you enable and does not send data off your device.

Data stored locally

The extension stores these preferences in chrome.storage.local:

  • Whether the extension is on.
  • Site origins where RTL Mode is enabled.
  • Site origins where Force RTL Layout is enabled.
  • Popup theme preference: System, Light, or Dark.

The extension does not use chrome.storage.sync.

Page text

The content script reads page text in memory to detect Hebrew, Arabic, and other right-to-left characters. It uses that result to set standard HTML dir attributes on matching elements.

The extension does not save, copy, transmit, or sell page text.

Network and third parties

The extension has no accounts, analytics, cookies, remote database, external API, or third-party service. It makes no network requests.

Retention

Local preferences stay on your device until you change them, clear extension data in Chrome, or uninstall the extension.
